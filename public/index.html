Northcoders News

API Endpoints

GET /api/topics

get all the topics

GET /api/topics/:topic/articles

return all the articles for a certain topic id

GET /api/articles

get all the articles

GET /api/articles/:article_id/comments

get all the comments for an individual article

POST /api/articles/:article_id/comments

add a new comment to an article

PUT /api/articles/:article_id

increment or decrement the votes of an article by one. A vote of query 'up' or 'down'

PUT /api/comments/:comment_id

increment or decrement the votes of an article by one. A vote of query 'up' or 'down'

DELETE /api/comments/:comment_id

deletes a comment if the comment was created by a Northcoder user

GET /api/users

get all users

GET /api/users/:username

returns an object with information for a person with a specific user ID
